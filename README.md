# UK Postcode Lookup 🏠

Free UK postcode to address lookup. Enter any UK postcode and get the full address, county, parliamentary constituency, coordinates, and nearby postcodes.

Uses the [postcodes.io](https://postcodes.io) API — completely free, no API key needed.

## What It Does

- Enter a UK postcode and hit search
- Shows the full address breakdown (ward, district, county, region)
- Displays parliamentary constituency
- Shows latitude and longitude on a map link
- Lists nearby postcodes with distance

## How to Use

1. Download or clone this repo
2. Open `index.html` in your browser
3. Type a postcode (e.g. CH1 1SL) and press Search

No server needed. No API key needed. Just open and go.

## Tech

- Plain HTML, CSS, and JavaScript
- [postcodes.io](https://postcodes.io) API (free, open source)
- No dependencies, no build step
